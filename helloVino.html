Welcome to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages.

Basic Structure

  • DOCTYPE: Declares the document type.
  • HTML Tag: Wraps the entire HTML document.
  • Head: Contains metadata and links to resources.
  • Body: Contains the content displayed on the web page.

Example

        <!DOCTYPE html>
        <html>
        <head>
            <title>Page Title</title>
        </head>
        <body>
            <h1>Hello, World!</h1>
        </body>
        </html>
